3. A method that takes an array of integers, and returns a new array with every other element
Casual:
Given an array of integers

Iterate through each element
  - if element is divisible by 2
    - move element to a new array
  - otherwise,
    - go to the next iteration

After iterating through the array, return the new array

Formal:
START

# Given an array of integers called "numbers"

SET new_array = empty array
SET iterator = 1

WHILE iterator < length of numbers
  IF element of numbers % 2 = 0
    add element to new_array

  iterator = iterator + 1
  
PRINT new_array

END
